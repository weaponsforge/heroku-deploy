## heroku-deploy

> More tests on ci/cd to heroku.


## Usage

1. Clone this repository.  
`git clone https://github.com/weaponsforge/heroku-deploy`

2. Install dependencies.  
`npm install`

3. Run local express server.  
`npm run start`

3. Load localhost website on a web browser.  
`http://localhost:3000`
